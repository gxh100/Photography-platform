# Photography-platform
# 📸 摄影平台

> 一个基于 Spring Boot + Vue 的摄影作品展示、预约拍摄与轻量级社交平台。  
> 摄影师可入驻展示作品，用户可浏览、收藏、预约拍摄服务，支持在线沟通与作品评论。
---
## ✨ 核心功能

### 用户端
- **注册/登录**：手机号或邮箱注册，JWT 认证
- **作品浏览**：按分类、标签、热度、时间排序查看摄影作品
- **作品详情**：查看高清图、摄影师信息、EXIF 数据、点赞/收藏/评论
- **摄影师主页**：展示摄影师简介、作品集、服务套餐
- **预约拍摄**：选择套餐、日期、备注，提交预约申请
- **个人中心**：管理收藏、预约记录、个人资料

### 摄影师端
- **摄影师入驻申请**：提交资质审核
- **作品管理**：上传、编辑、下架摄影作品
- **预约管理**：查看/处理/确认用户的预约请求
- **服务套餐管理**：发布不同价位的拍摄套餐
- **数据统计**：查看作品浏览量、获赞数、预约转化率

### 管理后台（管理员）
- **用户管理**：禁用/启用用户，重置密码
- **摄影师审核**：审核入驻申请
- **内容审核**：下架违规作品或评论
- **系统配置**：首页轮播、公告、分类管理
---
## 🛠 技术栈

| 前端 | 后端 | 数据库/其他 |
|------|------|------------|
| Vue 2 / Vue 3 | Spring Boot 2.7 | MySQL 8.0 |
| Element UI  | MyBatis-Plus 3.5 | Redis（缓存/验证码） |
| Axios | JWT 认证 | MinIO / 阿里云OSS（图片存储） |
| Vue Router | Spring Security | Swagger / Knife4j（API 文档） |
